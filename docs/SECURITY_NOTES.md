# Security notes

## Review date
2026-07-13 (`chore/portfolio-quality-pass`)

## Findings
- No application secrets are required for the public lab.
- Local files `.env.local` and `.vercel/.env.production.local` may contain Vercel OIDC/session material generated by the CLI.
- These paths are covered by `.gitignore` (`.env.local`, `.env.*.local`, `.vercel/`).
- `.env.example` contains **placeholders only**.

## Actions taken
- Verified ignore rules before staging.
- Did not print or commit secret values.
- Confirmed seed data is synthetic (no real client/patient PII).

## Residual guidance
- Never force-add `.env*` or `.vercel/`.
- Do not store clinical/medical data in this lab.
- Rotate any token if it was ever pasted into chat logs or screenshots.
